# User Manager System

The **User Manager System** is a web-based application that simulates a user management module for a banking platform.  
The system is architected with **role-based access control**, separating functionality between **Client** and **Administration** portals to ensure security and data integrity.

## System Overview

The application provides authentication, authorization, and user data management features, allowing users to securely manage their personal information while enabling administrators to perform full CRUD operations on client accounts.

## Functional Features

### Client Portal
- User registration and authentication  
- Read and update personal user data  
- Secure password update  

### Admin Portal
- Administrator authentication  
- Read and update administrator profile data  
- List all client users  
- Read and update client user data  
- Delete client user accounts  
- Search and filter users  

## Security & Access Control
- Role-based access restriction between Client and Admin portals  
- Secure password storage using hashing  
- Unauthorized access prevention between system roles  

## Project Scope
- Separation of concerns between user roles  
- Implementation of CRUD operations  
- Focus on security best practices for authentication and authorization
