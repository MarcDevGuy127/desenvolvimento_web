
# Projeto WEB - Site de Cinema

Durante a disciplina de Programação WEB, foi proposto desenvolver um site que simulasse um sistema de gerenciamento de usuário. Nele simulamos cadastros, logins, edições e exclusões de dados com ambientes restritos para Administradores e Usuários.

# O que foi feito?
Durante o desenvolvimento deste projeto, foram implementadas as seguintes duncionalidades:

## Usuários:
- Cadastro/Login
- Consulta/Edição de dados cadastrais
- Troca de senha
## Administradores
- Cadastro/Login
- Consulta/Edição de dados cadastrais
- Troca de senha
- Lista de usuários cadastrados
- Edição de dados dos usuários
- Exclusão de usuários
- Ferramenta de pesquisa e filtros
